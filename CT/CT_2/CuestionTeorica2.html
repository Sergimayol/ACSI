Si en vez de usar un servidor para ejecutar un programa se usa otro servidor distinto, supongamos que se mejora el rendimiento con ese segundo servidor, ya que parece que el programa ahora se ejecuta en menos tiempo. Pero también sospechamos que se usa más la CPU que en el primero. ¿Cómo plantearías la comprobación de la mejora de un servidor sobre el otro? ¿Y de ese mayor uso de CPU? ¿Cómo podrías establecer el rendimiento de ambos servidores para que la comparación fuese justa?

Para comprobar la mejora de un servidor sobre el otro, si obtenemos el tiempo de ejecución de cada servidor y realizamos una división entre ambos, logramos obtener el % de mejora de uno sobre el otro. En este caso, si dividimos el tiempo del primer servidor entre el segundo, se obtiene cuanto mejora el segundo sobre el primero.

La comprobación de un mayor uso de la CPU, se puede obtener monitorizando ambas CPUs, ya sea vía hardware, software o un híbrido entre ambas, y ver el % de uso en cada servidor. La manera más sencilla sería con una monitorización vía software.

Para que fuese justa la comprobación en ambos casos, se tendrían que monitorizar en las mismas condiciones, misma carga y medir su rendimiento cuando se estén empleando al 100%.
