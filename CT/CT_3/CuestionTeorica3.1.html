¿Son equivalentes las medidas de rendimiento de los tiempos de respuesta de los cuatro sistemas (A,B,C,D) utilizando los tres tipos de medias? ¿Por qué?

Si calculas las aceleraciones de los programas sobre el sistema de referencia R, ¿Qué tipo de media harías para obtener una medida de rendimiento correcta?

Si en vez de calcular los valores normalizados por filas (aceleraciones individuales de los programas en cada sistema sobre R), se utiliza la aceleración suma de tiempos totales de los sistemas (por columnas), ¿qué media estarías empleando?

Y si normalizamos las medias aritméticas de los tiempos, ¿qué media usas para la máquina de referencia? ¿Por qué?

Como se puede observar trás los cálculos las medidas de rendimiento de los sistemas utilizando los tres tipos de media son diferentes. Las medidas no son equivalentes, debido a que cada media se utiliza para obtener un índice distinto dependiendo de las medidas tomadas, es decir, cada media tiene una mayor utilidad cuando se tienen los datos específicos para emplear ese promedio.

Para calcular la media y obtener una medida de rendimiento correcta, en cuanto a las aceleraciones, la media más adecuada sería la geométrica. Este tipo de media se utiliza  para calcular la tasa de crecimiento porcentual promedio, y los datos obtenidos trás el cálculo de es la mejora respecto al sistema de referencia, es decir, el porcentaje que se ha mejorado, además de obtener los resultados ya normalizados.

Si se utiliza la aceleración suma de tiempos totales de los sistemas, se estaría empleando la media aritmética del tiempo de respuesta de cada sistema sin normalizar .

La media a utilizar sería la misma que con los valores normalizados, la media aritmética,  ya que no tendría sentido comparar las media calculadas con diferentes métodos.
